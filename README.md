# This is a test harness project of OpenBLAS for RISC-V.
#
# It handles BF16, FP32 and FP64 for GEMM and GEMV.
# It handles real-only data.
# It handles non-transposed and tranposed data (via packing).
# It tests correctness and performance.
#
# It is ultimately designed to show improvements to kernels.
